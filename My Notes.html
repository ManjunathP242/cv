TABLE

  • Use table<> tag to create tables.
  • Use tr<> tag inside table<> to create rows in a table.
  • Use td<> to add to a cell inside tr<>.
  • Tables has three sections namely Header,Body,Footer.
  • Use thead<> for table header, add row using tr<> and add header data using th<>.
  • Use tbody<> for table body, add row using tr<> and add body data using td<>.
  • Use tfoot<> for table footer, add row using tr<> and add footer data using th<> or td<>.
  • Use cellspacing="20" inside start tag of table to give spacing.
